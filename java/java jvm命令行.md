### Java JVM启动参数

```
java [options] classname [args]
java [options] -jar filename [args]

options: 命令行选项，空格分隔
classname: 启动的class名称
filename: Jar包名称(JAR)
args: main()方法参数，空格隔开
```

### JVM参数
```
-Xms    最小堆大小，如：-Xms512m
-Xmx    最大堆大小，如：-Xmx1024m

-Xss    每个线程的堆栈大小，如：-Xss1024k

```
